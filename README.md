# Weather Dashboard

A real-time weather application that displays current weather conditions for any city using the OpenWeatherMap API.

## Features
- Real-time weather data retrieval
- Location-based forecasting
- In-memory caching to reduce API calls
- Responsive design
- Clean, modern UI

## Technologies Used
- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **API:** OpenWeatherMap
- **Deployment:** Render

## How to Run Locally
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Add your OpenWeatherMap API key to `app.py`
4. Run: `python app.py`
5. Open browser to `http://localhost:5000`
